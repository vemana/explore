# What is this repo?

An eclectic mix of small projects for one of
- my own amusement
- learning new things by doing small projects
- sharing knowledge and occasionally wisdom


# What's in there?
- **anthropic_original_performance_takehome**: This is Anthropic's performance challenge for optimizing instruction packing into a VLIW machine. I wrote a custom a programming language, a compiler targeting the VLIW machine and an optimizer.

- **galtonboard**: Simulates a Galtonboard. hosted at [https://galtonboard.pages.dev/](https://galtonboard.pages.dev/)

- **cuda_bellmanford**: A simple project to (1) explore CUDA and (2) try out optimizing memory bandwidth of GPUs

- **fast_cpp_spsc_queue**: A fast SPSC queue (2.2B int64 exchanges/sec on Ryzen 7950X) in C++ that is in sympathy with the hardware. Modern processors (even CPUs) are remarkably fast if you use them in a certain way!

- **1brc**: The (in)famous Java [1-Billion Row Challenge](https://github.com/gunnarmorling/1brc). This repo contains my entry and helper scripts to run it (if it still runs!). It finished top 20 in the comp.
